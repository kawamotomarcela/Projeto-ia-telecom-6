#  Premissas e Limitações

- O sistema será desenvolvido inicialmente como um protótipo acadêmico.
- A base de dados utilizada pode ser limitada em tamanho ou diversidade.
- O modelo de Machine Learning pode apresentar margem de erro nas previsões.
- A precisão das previsões depende da qualidade e da quantidade de dados históricos disponíveis.
- O sistema não substitui a tomada de decisão humana, servindo apenas como ferramenta de apoio para planejamento e gestão das Ordens de Serviço.
- A solução não contempla integração direta com sistemas reais de gestão de serviços, sendo voltada inicialmente para fins educacionais e experimentais.
